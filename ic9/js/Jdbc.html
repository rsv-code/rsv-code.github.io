Class: Jdbc

Jdbc

new Jdbc(Driver, Url, UserName, Password)

Jdbc object implements generic Java JDBC functionality. This object can be extended to implement more specific connectors or simply be used as is to manage a database connection. Object constructor takes connection information and creates a new JDBC object.
Parameters:
Name Type Description
Driver is a string with the JDBC driver.
Url is a string with the database url to connect to.
UserName is a string with the user name to connect with.
Password is a string with the password to connect with.
