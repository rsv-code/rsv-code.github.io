Timer

new Timer(OnTimeout, TimeoutMills)

Timer object runs a function at the provided timeout period or can be set to run the function on an interval.
Parameters:
Name Type Description
OnTimeout is a function to call on timeout.
TimeoutMills is an integer with the timer delay.
