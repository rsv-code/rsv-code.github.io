Class: Ldap

Ldap

new Ldap(Host, Root, UserName, Password, Port)

Default constructor takes the LDAP configuration information. If the user name contains a forward slash as used with Windows Active Directory (ex: jsmith/somecompany) then the root parameter is unused and the user name is used by iteslf. If not then the user name is a combination of the user name and the root (ex: cn=[UserName],[Root]).

So if the bind DN is: cn=read-only-admin,dc=example,dc=com you would pass the following.
UserName = 'read-only-admin'
Root = 'dc=example,dc=com'

For active directory with user jsmith at domain somecompany you would pass the following.
UserName = 'jsmith/somecompany'
Root = ''

Host Is a String with the LDAP/AD host to connect to.
Root Is a String with the root part of the DN or blank.
UserName Is a String with the user name.
Password Is a String with the password.
Port Is an int with the host port number to connect to. (Optional, default is 389.)
